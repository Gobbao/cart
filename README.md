# cart
Cart sample using vue and vuex

## Installation
```
$ git clone https://github.com/Zupter/cart.git
$ cd cart
$ npm install
```

Create a `config.local.js` file on `src/config` directory using `config.example.js` as example.

## Usage
```
$ npm run dev
```

## Tests
```
$ npm run tests
```
